¿Qué es?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Se destaca por su sintaxis sencilla y legible, lo que lo convierte en una excelente opción tanto para principiantes como para profesionales. Su diseño busca que el código sea fácil de escribir y entender, pareciéndose más al lenguaje humano que a la estructura técnica de otros lenguajes.

Además, Python es un lenguaje multiparadigma, lo que significa que permite distintos estilos de programación: estructurada, orientada a objetos y funcional. Gracias a esto, puede utilizarse en una gran variedad de campos como inteligencia artificial, desarrollo web, análisis de datos, automatización y más.


¿Cuál es su función?

El propósito principal de Python es facilitar la creación de programas claros, eficientes y versátiles sin necesidad de usar una sintaxis complicada.
Sus funciones más destacadas son:

  • Automatizar tareas repetitivas (por ejemplo, mover archivos, procesar datos o generar reportes).
  • Desarrollar aplicaciones web y de escritorio, usando frameworks como Django o Flask.
  • Trabajar con inteligencia artificial y análisis de datos, gracias a librerías potentes como TensorFlow, Pandas y NumPy.

Además, Python tiene una gran comunidad mundial que constantemente desarrolla librerías nuevas, lo cual amplía sus capacidades. Por su facilidad de aprendizaje y su enorme ecosistema, es uno de los lenguajes más usados y solicitados en el mundo de la programación.


¿Qué sabemos de su historia?

Python fue creado por Guido van Rossum a finales de los años 80, en el Centro para las Matemáticas y la Informática (CWI) en los Países Bajos. Su desarrollo comenzó en 1989 y la primera versión oficial se lanzó en 1991. Guido buscaba un lenguaje que fuera tan poderoso como C, pero mucho más fácil de leer y escribir. De hecho, el nombre “Python” no viene del animal, sino del grupo de comedia británico Monty Python, ya que su creador quería un lenguaje divertido de usar.

Con el tiempo, Python evolucionó con versiones como Python 2.0 (2000) y Python 3.0 (2008), esta última marcando un cambio importante que mejoró la claridad y la coherencia del lenguaje. Hoy en día, Python 3 es el estándar mundial y continúa recibiendo actualizaciones.

Actualmente, Python es uno de los lenguajes más populares del planeta, usado por empresas como Google, NASA, Meta y Netflix. Su combinación de simplicidad, potencia y comunidad activa lo ha convertido en una herramienta esencial para la tecnología moderna.


¿Cuales son sus etiquetas basicas?

Aunque el lenguaje de Python tiene un amplio y profundo contenido en cada una de sus librerias, por ahora solo hablaremos de la categoría más básica de este.

  • print()
  • input()
  • if/elif/else
  • for
  • while
  • def
  • import
  • class
  • try/except
  • return

print()

Significado: “Imprimir”

Explicación: Se utiliza para mostrar información en pantalla. Puede imprimir texto, números, resultados de operaciones o el contenido de variables. Es una de las primeras funciones que se aprende en Python, porque permite visualizar lo que el programa está haciendo.

input()

Significado: “Entrada”

Explicación: Permite recibir datos del usuario desde el teclado. Todo lo que el usuario escriba se almacena como texto (tipo str), aunque luego se puede convertir a número si se necesita.

if/elif/else

Significado: “Si / sino si / de lo contrario”

Explicación: Se usan para tomar decisiones dentro del programa. Evalúan condiciones y ejecutan diferentes bloques de código según si son verdaderas o falsas.

for

Significado: “Para”

Explicación: Es un bucle que recorre una secuencia (lista, cadena, rango, etc.) y ejecuta el código dentro de él por cada elemento. Es muy usado para procesar datos o repetir acciones un número conocido de veces.

while

Significado: “Mientras”

Explicación: Ejecuta un bloque de código mientras una condición sea verdadera. Se usa cuando no se sabe cuántas veces se repetirá la acción, por ejemplo, al esperar una entrada específica del usuario.

def

Significado: “Definir”

Explicación: Se usa para definir una función, es decir, un bloque de código que realiza una tarea específica y puede reutilizarse. Las funciones pueden recibir valores (parámetros) y devolver resultados.

import

Significado: “Importar”

Explicación: Sirve para traer módulos o librerías externas a tu programa. Estas librerías contienen funciones ya creadas que amplían las capacidades de Python, como trabajar con matemáticas, fechas o archivos.

class

Significado: “Clase”

Explicación: Define una clase, que es una plantilla para crear objetos. Se usa en la programación orientada a objetos para agrupar datos (atributos) y comportamientos (métodos) relacionados.

try/except

Significado: “Intentar / excepto”

Explicación: Se usa para manejar errores o excepciones. Permite que el programa no se detenga si ocurre un error, sino que ejecute una acción alternativa.

return

Significado: “Devolver”

Explicación: Se utiliza dentro de una función para enviar un resultado o valor al lugar desde donde fue llamada. Es la forma de que una función “entregue” su resultado.
