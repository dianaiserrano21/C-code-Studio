¿Qué es?

MySQL es un sistema de gestión de bases de datos relacional (RDBMS) basado en el lenguaje SQL (Structured Query Language). Permite almacenar, organizar y acceder a grandes volúmenes de información de forma estructurada, utilizando tablas que se relacionan entre sí. Es una de las herramientas más populares en el desarrollo web y empresarial, gracias a su velocidad, confiabilidad y facilidad de uso.

Además, MySQL es software libre y de código abierto, lo que significa que puede ser utilizado y modificado sin costo. Por esta razón, es muy común encontrarlo integrado en servidores web junto con lenguajes como PHP o Python, formando parte del famoso paquete LAMP (Linux, Apache, MySQL y PHP).


¿Cuál es su función?

La función principal de MySQL es gestionar la información de forma segura, organizada y accesible dentro de una base de datos. Permite que los programas puedan guardar datos, consultarlos, actualizarlos o eliminarlos cuando sea necesario.
Entre sus funciones más importantes se encuentran:

  • Crear y administrar bases de datos que almacenan información estructurada.
  • Ejecutar consultas SQL para buscar o modificar registros con rapidez y precisión.
  • Controlar el acceso a los datos, estableciendo usuarios, permisos y contraseñas.

Gracias a estas capacidades, MySQL se usa en miles de aplicaciones y sitios web que requieren manejar datos en tiempo real: redes sociales, tiendas en línea, sistemas escolares o bancarios. Su eficiencia permite trabajar con millones de registros sin perder rendimiento.


¿Qué sabemos de su historia?

MySQL fue desarrollado en 1995 por Michael Widenius, David Axmark y Allan Larsson, como un proyecto destinado a crear una alternativa gratuita y rápida a los gestores de bases de datos comerciales. Su nombre proviene de “My” (el nombre de la hija de Widenius) y “SQL”, el lenguaje que utiliza.

Durante los años 2000, MySQL creció enormemente gracias a su inclusión en el entorno LAMP, convirtiéndose en el corazón de muchos sitios web dinámicos. En 2008, la empresa Sun Microsystems compró MySQL AB (la compañía detrás del proyecto), y poco después, en 2010, Oracle Corporation adquirió Sun, quedando como el principal responsable de su desarrollo.

Aun así, MySQL sigue siendo uno de los sistemas de bases de datos más usados del mundo, valorado por su estabilidad, velocidad y compatibilidad con casi todos los lenguajes de programación. Su historia refleja el equilibrio entre la libertad del software abierto y el soporte empresarial que lo mantiene vigente hasta hoy.


¿Cuales son sus etiquetas basicas?

Aunque el lenguje de MySQL tiene un amplio y profundo contenido en cada una de sus librerias, por ahora solo hablaremos de la categoria más baja y basica de este.

  • CREATE DATABASE
  • USE
  • CREATE TABLE
  • INSERT INTO
  • SELECT
  • WHERE
  • UPDATE
  • DELETE
  • ALTER TABLE
  • DROP

CREATE DATABASE

Significado: “Crear base de datos”

Explicación: Se usa para crear una nueva base de datos donde se almacenará la información. Es el primer paso antes de crear tablas o insertar datos.
Ejemplo:

USE

Significado: “Usar”

Explicación: Permite seleccionar una base de datos específica con la que se va a trabajar. Una vez elegida, todos los comandos siguientes se aplican dentro de ella.
Ejemplo:

CREATE TABLE

Significado: “Crear tabla”

Explicación: Se utiliza para crear una tabla dentro de la base de datos. En ella se definen las columnas (campos) y el tipo de datos que almacenará cada una.

INSERT INTO

Significado: “Insertar en”

Explicación: Permite agregar registros (filas) nuevos a una tabla ya existente. Se indican las columnas y los valores que se insertarán.

SELECT

Significado: “Seleccionar”

Explicación: Se usa para consultar o visualizar los datos guardados en una tabla. Es el comando más usado en SQL, pues permite obtener información de distintas maneras.
Ejemplo:

WHERE

Significado: “Donde”

Explicación: Se usa junto con otros comandos (como SELECT, UPDATE, o DELETE) para filtrar los resultados que cumplan una condición específica.
Ejemplo:

UPDATE

Significado: “Actualizar”

Explicación: Sirve para modificar datos existentes en una tabla. Se indica qué campo cambiar y qué condición deben cumplir los registros a actualizar.

DELETE

Significado: “Eliminar”

Explicación: Se utiliza para borrar uno o varios registros de una tabla según una condición. Si no se pone WHERE, eliminará todos los registros.

ALTER TABLE
