¿Qué es?

C++ es un lenguaje de programación de propósito general, creado para desarrollar programas que van desde sistemas operativos hasta videojuegos y software de alto rendimiento. Se considera un lenguaje de nivel medio, ya que combina características de bajo nivel (como el manejo directo de memoria) con abstracciones de alto nivel, lo que permite escribir código eficiente sin perder legibilidad.

Además, C++ es una extensión del lenguaje C, y su principal diferencia radica en que agrega soporte para la programación orientada a objetos, lo que facilita organizar el código en clases y objetos, haciendo los programas más modulares y reutilizables. Por su potencia, se usa en aplicaciones donde la velocidad y el control del hardware son esenciales.


¿Cuál es su función?

El objetivo principal de C++ es desarrollar programas eficientes y rápidos que aprovechen al máximo los recursos del sistema.
Entre sus funciones más destacadas están:

  • Permitir la programación estructurada, orientada a objetos y genérica, dando libertad al programador de usar el enfoque que mejor se adapte a su proyecto.
  • Brindar control total sobre la memoria y los procesos, algo vital en el desarrollo de sistemas operativos, controladores y motores de videojuegos.
  • Servir como base para muchos otros lenguajes modernos (como Java o C#), que se inspiraron en su sintaxis y principios.

Gracias a esta versatilidad, C++ sigue siendo uno de los lenguajes más usados en ingeniería de software, simulaciones, inteligencia artificial, robótica y desarrollo de aplicaciones complejas donde se necesita estabilidad y rendimiento.


¿Qué sabemos de su historia?

C++ fue creado por Bjarne Stroustrup en 1983, mientras trabajaba en los laboratorios Bell de AT&T. Su intención era mejorar el lenguaje C añadiendo herramientas que facilitaran la programación de sistemas grandes y complejos. Al principio se llamó “C with Classes” (C con clases), ya que su principal innovación fue introducir la programación orientada a objetos, algo que C no tenía.

Con el tiempo, el lenguaje fue creciendo y añadiendo características como herencia múltiple, plantillas (templates) y manejo de excepciones. En 1998 se estandarizó oficialmente como ISO C++, lo que garantizó compatibilidad y estabilidad para los desarrolladores. Desde entonces, ha tenido varias versiones importantes (C++11, C++14, C++17, C++20 y C++23), cada una agregando mejoras en eficiencia, seguridad y facilidad de uso.

Hoy en día, C++ es considerado uno de los lenguajes más influyentes de la historia, base de sistemas como Windows, motores de videojuegos como Unreal Engine y software de alto rendimiento en empresas tecnológicas, científicas y financieras.


¿Cuales son sus etiquetas basicas?

Aunque el lenguaje de C++ tiene un amplio y profundo contenido en cada una de sus librerias, por ahora solo hablaremos de la categoría más básica de este.

  • #include
  • using namespace std;
  • int main()
  • {}
  • cout
  • cin
  • if/else
  • for
  • while
  • return

#include

Significado: "Incluir"

Explicación: Esta instrucción se usa para añadir librerías al programa antes de compilarlo. Las librerías contienen funciones y herramientas ya creadas, como las que permiten mostrar texto en pantalla o trabajar con archivos.

using namespace std;

Significado: “Usar el espacio de nombres estándar”

Explicación: Evita que tengas que escribir std:: antes de cada comando del lenguaje estándar. Por ejemplo, en lugar de escribir std::cout, simplemente puedes poner cout.
Sirve para simplificar el código y hacerlo más legible cuando se usan muchos elementos de la biblioteca estándar.

int main()

Significado: “Función principal del programa”

Explicación: Es el punto de inicio donde comienza a ejecutarse el programa. Siempre debe existir una función main() en cualquier código de C++, ya que sin ella el programa no sabría por dónde empezar.
El tipo int indica que la función devolverá un número entero al sistema operativo al finalizar.

{}

Significado: “Delimitadores de bloque de código”

Explicación: Se usan para encerrar el cuerpo de una función, un bucle o una condición. Todo lo que esté dentro de las llaves pertenece a ese bloque.

cout

Significado: “Character output” (salida de caracteres)

Explicación: Sirve para mostrar información en la pantalla. Se usa junto con el operador << para enviar texto o valores a la consola.

cin

Significado: “Character input” (entrada de caracteres)

Explicación: Se utiliza para leer datos que el usuario escribe en la consola. Se combina con el operador >>.

if/else

Significado: “Si / de lo contrario”

Explicación: Permite ejecutar diferentes partes del código dependiendo de si una condición es verdadera o falsa.

for

Significado: “Para” (bucle controlado)

Explicación: Se usa para repetir un bloque de código un número determinado de veces, generalmente cuando se sabe cuántas repeticiones se necesitan.

while

Significado: “Mientras”

Explicación: Ejecuta un bloque de código mientras una condición sea verdadera. Es ideal cuando no se sabe cuántas repeticiones se harán.

return

Significado: “Devolver”

Explicación: Explicación: Indica que la función ha terminado su ejecución y devuelve un valor al sistema operativo o a la función que la llamó.



Características de C++

Características Descripción
Lenguaje C++ es un lenguaje de programación de propósito general, rápido y eficiente.
Paradigma Soporta programación estructurada, orientada a objetos y genérica.
Compilado El código se compila antes de ejecutarse, lo que lo hace más rápido.
Fuertemente Tipado Cada variable necesita un tipo claro y coherente. Nada de improvisaciones.
Manejo de memoria El programador controla la memoria con new, delete y punteros.
Portabilidad Funciona en Windows, Linux, macOS y más, si tienes un compilador competente.
Librerías estándar Tiene STL, con vectores, listas, mapas, algoritmos… todo para no sufrir tanto.
Extención de C Está basado en C, pero con muchas mejoras para trabajar con objetos.
Uso común Desarrollo de videojuegos, sistemas embebidos, software de alto rendimiento.
Performance Destaca por ser extremadamente rápido y eficiente en recursos.
