¿Qué es?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Su función principal es estructurar el contenido de un sitio mediante etiquetas que indican qué tipo de información se está mostrando, como textos, imágenes, enlaces o tablas. No es un lenguaje de programación, sino un lenguaje de marcado, lo que significa que se utiliza para organizar y presentar información, no para ejecutar procesos lógicos o cálculos.

Además, HTML es la base de toda página web. Todo lo que ves en internet, desde un simple texto hasta sitios complejos, tiene una estructura creada con este lenguaje. Los navegadores (como Chrome o Firefox) leen las etiquetas HTML y las interpretan para mostrar visualmente la información de manera ordenada y atractiva al usuario.


¿Cuál es su función?

  • La función principal de HTML es definir la estructura del contenido de una página web. Cada elemento —párrafos, títulos, listas, imágenes o enlaces— se marca con etiquetas específicas que le indican al navegador cómo mostrarlo.
  • También permite enlazar documentos y recursos mediante hipervínculos, que conectan una página con otra dentro o fuera de un sitio web. Esto da forma a la red de información interconectada que conocemos como internet.
  • Por último, HTML trabaja en conjunto con otros lenguajes, como CSS (para el diseño visual) y JavaScript (para la interactividad). HTML construye la base o el esqueleto del sitio; CSS lo viste con estilos; y JavaScript le da vida y dinamismo.

¿Qué sabemos de su historia?

HTML fue creado a principios de los años 90 por Tim Berners-Lee, un científico del CERN, con el propósito de facilitar el intercambio de información entre investigadores a través de la red. La primera versión, HTML 1.0, apareció en 1993 y era muy básica, permitiendo solo texto, enlaces e imágenes simples.

Con el tiempo, el lenguaje fue evolucionando: HTML 2.0 (1995) estandarizó las primeras funciones; HTML 3.2 (1997) añadió tablas y soporte para scripts; HTML 4.01 (1999) mejoró la organización del contenido y la accesibilidad; y finalmente HTML5, lanzado oficialmente en 2014, se convirtió en la versión moderna más usada, integrando soporte para video, audio, gráficos y aplicaciones interactivas sin necesidad de complementos externos.

Hoy, HTML sigue siendo el pilar del desarrollo web, y su evolución continúa, adaptándose a las nuevas tecnologías y dispositivos para mantener la web accesible y funcional en todo el mundo.


¿Cuales son sus etiquetas basicas?

Aunque el lenguje de HTML tiene un amplio y profundo contenido en cada una de sus librerias, por ahora solo hablaremos de la categoria más baja y basica de este.

  • html
  • head
  • title
  • body
  • h1 a h6
  • p
  • a
  • img
  • div
  • span

html

Significado: “Raíz del documento”

Explicación: Esta etiqueta marca el inicio y fin de un documento HTML. Todo el contenido de la página web debe ir dentro de ella. Es la estructura principal que indica al navegador que el código que contiene es HTML.

head

Significado: “Cabecera, donde va la configuración”

Explicación: Dentro de head se colocan los metadatos, enlaces a hojas de estilo, scripts, y el título de la página. Esta parte no es visible para el usuario directamente, pero es fundamental para el funcionamiento y presentación del sitio.

title

Significado: “Título de la página”

Explicación: Define el nombre que aparece en la pestaña del navegador y en los resultados de búsqueda. Es importante para la identificación del sitio y también para el posicionamiento en buscadores.

body

Significado: “Contenido visible”

Explicación: Todo lo que el usuario ve en una página web (texto, imágenes, botones, enlaces, etc.) se coloca dentro de body. Es el cuerpo principal donde se muestra la información.

h1 a h6

Significado: “Encabezados”

Explicación: Son etiquetas de títulos que van desde h1 (más importante y grande) hasta h6 (más pequeño y menos importante). Ayudan a organizar el contenido y mejorar la accesibilidad y estructura del texto.

p

Significado: “Párrafo”

Explicación: Sirve para escribir bloques de texto o párrafos. Se utiliza para separar y estructurar el contenido de forma legible dentro de una página.

a

Significado: “Enlace”

Explicación: Crea hipervínculos hacia otras páginas, archivos o secciones del mismo documento. Se usa con el atributo href para indicar la dirección del enlace.

img

Significado: “Imagen”

Explicación: Permite insertar imágenes en la página web. Usa el atributo src para definir la ruta de la imagen y alt para agregar una descripción alternativa, útil en temas de accesibilidad.

div

Significado: “Contenedor genérico”

Explicación: Se usa para agrupar otros elementos y aplicarles estilos o scripts. No tiene significado propio, pero es muy útil para estructurar y organizar el diseño de la página.

span

Significado: “Texto en línea o pequeños fragmentos.”

Explicación: Sirve para aplicar estilos o cambios a partes pequeñas del texto sin afectar el resto del contenido. Es una etiqueta en línea, ideal para modificar palabras o frases específicas.
